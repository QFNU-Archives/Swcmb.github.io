```python
n=int(input())

li=list(map(int,input().split()))

max=-1

def distance(i, j):

  return abs(i-j)+abs(li[i]-li[j])

for i in range(len(li)):

  for j in range(1,len(li)):

     if distance(i,j)>max:

       max=distance(i,j)

print(max)
```
